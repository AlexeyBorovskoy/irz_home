# IRZ_HOME_WIFI_LAB

Инженерный проект аудита, оптимизации и расширения домашней Wi-Fi инфраструктуры.

---

## Назначение

Систематический аудит существующей Wi-Fi сети, выявление проблем, оптимизация параметров и подготовка к развёртыванию расширителя/репитера на базе iRZ RL22w.

---

## Оборудование

| Устройство       | Роль                        |
|------------------|-----------------------------|
| TP-Link EC220-G5 | Основной роутер / точка AP  |
| iRZ RL22w        | Репитер / мост (планируется)|
| Windows Host     | Рабочая станция             |
| VMware Workstation | Гипервизор               |
| Linux Mint VM    | Инструментальная среда      |

---

## Архитектура (текущая)

```
Internet
   │
TP-Link EC220-G5 (192.168.0.1)
   │
Windows Host
   │
VMware Workstation
   │
Linux Mint VM
```

## Архитектура (планируемая)

```
Internet
   │
TP-Link EC220-G5
   ))))  (Wi-Fi)
iRZ RL22w
   │
Extended Wi-Fi Zone
```

---

## Цели

1. Аудит текущей конфигурации роутера и Wi-Fi параметров
2. Выявление проблем: каналы, режимы, DHCP, безопасность
3. Инвентаризация сети
4. Подготовка к интеграции iRZ RL22w
5. Развёртывание репитера/моста/STA+AP

---

## Ограничения

- Все деструктивные операции — только после явного подтверждения
- Сначала аудит, потом изменения
- Подробнее: [SECURITY_RULES.md](SECURITY_RULES.md)

---

## Этапы

1. ✅ Создание структуры проекта
2. 🔄 Инвентаризация оборудования
   - ✅ Linux VM — сеть, интерфейсы
   - ✅ iRZ RL22w — полный аудит через API (2026-05-09)
   - ⬜ TP-Link EC220-G5 — требует web UI
3. 🔄 Аудит сети и Wi-Fi
   - ✅ iRZ — конфигурация получена и сохранена
   - ⬜ TP-Link — канал, DHCP, режимы
   - ⬜ Спектральный анализ Wi-Fi
4. ⬜ Анализ TP-Link (скриншоты web UI)
5. 🔄 Подготовка iRZ к интеграции
   - ⬜ Настройка APN (LTE)
   - ⬜ Смена часового пояса
   - ⬜ Выбор режима: AP / STA+AP / WDS
   - ⬜ Разнесение Wi-Fi каналов с TP-Link
6. ⬜ Развёртывание и тестирование
