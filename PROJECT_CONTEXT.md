# PROJECT_CONTEXT

## Текущая инфраструктура

### Windows Host
- Основная рабочая станция
- VMware Workstation установлен на хосте
- Подключён к TP-Link EC220-G5 по LAN или Wi-Fi

### Linux Mint VM
- Виртуальная машина под управлением VMware Workstation
- Инструментальная среда: nmcli, iw, iwlist, nmap, wireshark
- Сетевой адаптер: bridged или NAT (уточняется в ходе аудита)

### VMware Workstation
- Гипервизор на Windows Host
- Режим сети VM: подлежит уточнению

### TP-Link EC220-G5
- Основной домашний роутер
- LAN IP: 192.168.0.1
- Режимы Wi-Fi: 2.4 GHz + 5 GHz (уточняется)
- DHCP сервер: активен (параметры уточняются)
- Текущие каналы: подлежат аудиту

### iRZ RL22w
- Промышленный LTE-роутер с Wi-Fi
- Планируется как репитер / WDS-мост / STA+AP
- Текущее состояние: не подключён, ожидает настройки

### VPN
- Наличие и тип VPN: подлежит уточнению в ходе аудита

## Планы

### Краткосрочные
- Провести полный аудит текущей Wi-Fi и LAN конфигурации
- Зафиксировать текущие настройки TP-Link
- Оценить зону покрытия

### Среднесрочные
- Интегрировать iRZ RL22w как расширитель зоны
- Выбрать оптимальный режим: Repeater / WDS / STA+AP
- Протестировать throughput и roaming

### Долгосрочные
- Настроить мониторинг сети
- Документировать финальную конфигурацию
